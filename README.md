# DINORUN for Arduino
このプロジェクトは１時間以内に作れました。\
DINORUNで、ジャンプしか出来ません。

## 必要なハード
* Arduino Uno R3
* Grove Starter Kit（ベースシールド、LCD、ボタン、バザー）

## 必要なソフト
* Arduino IDE
